# Rainar CLI

A powerful, enterprise-grade project scaffolding tool.

## Project Vision

For a detailed explanation of the project's vision, goals, and methodology, please see the [Project Vision](./VISION.md) document.

## Installation

```bash
npm install -g @rainar/cli
```

## Usage

```bash
rainar new <project-name>
```

This will create a new project in a directory with the specified name. The project will be populated with a production-ready Node.js template.
